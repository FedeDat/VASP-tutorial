# VASP-tutorial
 VASP course given at the Institute of Chemical Research of Catalonia during September 2021, consisting of:
 (1) First class covering VASP handson 1;
 (2) Second class covering VASP handson 2; 
 (3) Third class covering VASP handson 3;
 (4) Fourth class covering VASP handson 4;
 (5) Additional class covering an introduction to Blender.
